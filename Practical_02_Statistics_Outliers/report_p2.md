# Practical 02 - Statistics and Outliers

Placeholder report for the statistics/outliers practical.

## 1. Which column had the highest data quality risk?
Answer:

The Order Date column had the highest data quality risk because incorrect or inconsistent date formats can affect time-based analysis and reporting.

## 2. What cleaning decision did you take and why?

Answer:

I removed duplicate records, checked for blank values, corrected data types (dates, numbers, and currency), and formatted the data consistently. These steps improve data accuracy and make the analysis more reliable.

## 3. Why should raw data and cleaned data be kept separately?

Answer:

Raw data should be kept unchanged as the original source. Cleaned data is used for analysis after fixing errors. Keeping them separate allows us to compare changes, avoid losing original information, and repeat the cleaning process if needed.
