# Data Dictionary - Superstore Sales Dataset

This dataset contains retail sales transaction data.

| Column Name     | Data Type | Meaning |
|----------------|----------|--------|
| Row ID         | Integer  | Unique row identifier |
| Order ID       | String   | Unique order number |
| Order Date     | Date     | Date when order was placed |
| Ship Date      | Date     | Date when order was shipped |
| Ship Mode      | String   | Shipping method used |
| Customer ID    | String   | Unique customer ID |
| Customer Name  | String   | Name of customer |
| Segment        | String   | Customer segment (Consumer/Corporate/Home Office) |
| Country        | String   | Country of order |
| City           | String   | City of order |
| State          | String   | State of order |
| Postal Code    | Integer  | Postal code |
| Region         | String   | Region of sale |
| Product ID     | String   | Unique product ID |
| Category       | String   | Product category |
| Sub-Category   | String   | Product sub-category |
| Product Name   | String   | Name of product |
| Sales          | Float    | Sales amount |
