Placeholder report for the Power Query ETL practical.

## Report Question

## 1. How is Power Query different from manual Excel cleaning?

Answer:

Power Query automates data cleaning by saving all transformation steps, while manual Excel cleaning requires repeating the same steps every time new data is added. Power Query is faster, more accurate, and reduces human errors.

## 2. Which transformation would save time when new CSV data arrives?

Answer:

Saving transformations such as removing duplicates, handling blank values, changing data types, and renaming columns in Power Query would save time. When a new CSV file arrives, these steps can be applied automatically by refreshing the query instead of cleaning the data manually.
