## 1. Which pivot table gave the most useful insight?
Answer:
The Category-wise Sales pivot table gave the most useful insight because it clearly showed which product categories generated the highest sales and profit.

## 2. What changed when you used slicers?
Answer:
Slicers made the dashboard interactive by allowing the data to be filtered by category, region, or date. This made it easier to analyze specific parts of the dataset without changing the original data.

## 3. What is one limitation of doing this only in Excel?
Answer:
One limitation of Excel is that handling very large datasets and creating interactive dashboards is less efficient compared to tools like Power BI.
